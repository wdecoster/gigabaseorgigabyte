# Gigabase or gigabyte

Hugo-based blog about bioinformatics and long-read sequencing, available at [GitHub Pages](https://wdecoster.github.io/gigabaseorgigabyte/).  

## Local Development

```bash
/snap/bin/hugo server -D --bind 0.0.0.0
```

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch.

## Theme

This site uses [Hermit-V2](https://github.com/1bl4z3r/hermit-V2), a minimal and fast Hugo theme designed for bloggers.
